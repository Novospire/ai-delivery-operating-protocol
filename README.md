Repo-first AI-assisted delivery protocol for controlled, small-ticket development across ChatGPT, Claude Code, Codex, Antigravity and future AI coding tools.

This repository defines the general operating standard. It should not be copied wholesale into every project repository.

Each project should receive a short project-specific adapter.
